constructor

()

Defined in esnext/lib/interface.js:26

BaseInterface Constructor

Adds support for the done event while ensuring that errors are always handled correctly. It does this by listening to the error and completed events, and when the emit, we check if there is a done listener:

  • if there is, then emit the done event with the original event arguments
  • if there isn't, then output the error to stderr and throw it.
